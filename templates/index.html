Welcome to the Little Lemon Restaurant API! This API provides endpoints for managing menu items and table bookings.

API Endpoints

Authentication

POST /api/registration/
POST /api/login/

Menu Items

GET /api/menu/
POST /api/menu/
GET /api/menu/{id}/
PUT /api/menu/{id}/
DELETE /api/menu/{id}/

Table Bookings

GET /api/bookings/
POST /api/bookings/
GET /api/bookings/{id}/
PUT /api/bookings/{id}/
DELETE /api/bookings/{id}/

Authentication

All endpoints (except registration and login) require authentication using a token. Include the token in the Authorization header:

Authorization: Token your_token_here
